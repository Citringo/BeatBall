===============
== Beat Ball ==
===============

1. 概要

Beat Ball をダウンロードしていただき，ありがとうございます．

Beat Ball は，主に球技をモチーフとした音楽ゲームです．

現在は，テストプレイのみが可能となっています．

また，ご自身による譜面の制作が可能です．詳細な譜面仕様は， [BBF フォーマット](https://github.com/Citringo/BeatBall/wiki/BBFフォーマット)をご確認ください．
